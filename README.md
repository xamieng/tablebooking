# Online Table Booking API

## SET UP THE API

### 1. PREREQUISITE
- docker

### 2. BUILD
```
./gradlew build
```

### 3. START API
```
docker-compose -f docker-compose.yml up -d
```

### 3. STOP API
```
docker-compose -f docker-compose.yml down
```

### 5. API Specification
```
http://localhost:8080/swagger-ui/index.html#/
```
